In this project we will explore empirical probability using random samples. The two important functions to understand are sample and replicate. We use the sample function to set up our experiment, and replicate function to repeat it as many times as we want.

2 Coin Tosses

Recall the coin toss problem from HW2: Suppose Sarah and Ali toss two coins indenpendently \(n\) times, what is the probability that both of them have the same number of head

#while the 2 of them are tossing the coin n #times & considering that it is a fair coin such #that: prob(one head) =0.5
#there can be 0,1,2,...,n heads obtained by both them.
#A)
prob_same_no_of_head <- function(n){
  x<-0:n
  sum(dbinom(x,n,(1/2))*dbinom(x,n,(1/2)))
}
